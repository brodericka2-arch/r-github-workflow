# GitHub Workflow Practice (R)

This project is used to practice:

- Branching
- Pull Requests
- Code Review
- Merge Conflicts

## Setup

Open this project in Posit (RStudio).

## Files

- analysis.R → main script
- utils.R → helper functions

## Instructions

You will:
1. Create a branch
2. Make a change
3. Commit and push
4. Open a Pull Request
5. Review a peer’s work
6. Merge into main
